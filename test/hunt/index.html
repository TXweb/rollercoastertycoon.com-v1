WHATS THE IDEA?

The Treasure Hunt is your chance to win an exclusive RollerCoaster Tycoon mousemat (a rare item) by finding the special character icons hidden on this web site.

TO PLAY TREASURE HUNT

  1. Check the game rules to make sure you are allowed to play.
  2. Find all six game icons (shown below) which are hidden throughout the site. Each one has a code next to it.

  3. When you have all six codes, place them in the entry form.
  4. A prize draw will be held to select 5 winners from all the correct entries and the winners will receive a RollerCoaster Tycoon mousemat.
