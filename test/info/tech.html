GAME INFO
TECHNICAL SPECIFICATION

Platform: PC only

Minimum:
P90MHz, 16Mb RAM, 4XCD, 50MB hard disk, 1MB SVGA card

Recommended (for good game performance):
P200MHz, 32Mb RAM, 8XCD, 50MB hard disk, 2MB accelerated SVGA card

Ideal (for ultimate game performance):
PII 350MHz, 64Mb RAM, 8xCD, 180MB hard disk, 4MB accelerated SVGA card

Note:
All specs require Windows 95/98, Windows 95 compatible Sound Card, DirectX 5.0 (included on CD) and Mouse.
Does not need 3D support or accessories.
