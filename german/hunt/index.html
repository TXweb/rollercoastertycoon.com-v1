SCHATZSUCHE 5
WAS STECKT DAHINTER?

Bei der Schatzsuche haben Sie die Chance, ein exklusives RollerCoaster Tycoon Mauspad zu gewinnen. Dazu müssen Sie die speziellen Suchbilder finden, die in dieser Website versteckt sind.

SO GEHT ES

  1. Lesen Sie die Spielregeln damit Sie wissen, ob Sie mitspielen dürfen.
  2. Finden Sie die sechs Spiel-Icons (siehe unten), die irgendwo in der Website versteckt worden sind. Neben jedem finden Sie einen Code.

  3. Wenn Sie alle sechs Codes kennen, tragen Sie sie in das Teilnahme-Formular ein.
  4. Es werden aus allen richtigen Einsendungen 5 Gewinner ausgelost, und diese Gewinner erhalten ein RollerCoaster Tycoon Mauspad.
